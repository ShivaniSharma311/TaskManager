Task Manager Application

Description:
Spring Boot Task Manager application using CRUD operations.

Features:
- Add tasks
- View tasks
- Update tasks
- Delete tasks

Technologies:
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

API Endpoints:
GET /tasks
GET /tasks/{id}
POST /tasks
PUT /tasks/{id}
DELETE /tasks/{id}
